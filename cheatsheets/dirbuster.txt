dirbuster
web discovery

dirb url_base <wordlists_files> <options>

-a <user agent>         custom UA
-H <header>             custom header
-o <file>               output
-N <nf_code>            ignore responses with this http code
-r                      disable recursion
-R                      ask about which directories to recurse into
-u <user:pass>          basic auth
-x <extensions file>    search for these extensions
